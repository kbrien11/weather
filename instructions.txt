to  activate virtual env:

Windows:
1) cd backend
2) create virtual env
3) .\Scripts\activate 

Mac
1) cd backend
2) python3 -m venv env
3) source env/bin/activate



to start each folder, backend and frontend:


backend
1) cd into backend folder
2) pip install -r requirements.txt -- requirements should be there though
3) python controller.py -- This should start the Flask app

frontend - new terminal window

1) cd frontend
2) npm install 
3) npm start
